# 0x00. C - Hello, World

## Description

This project introduces the fundamentals of the C programming language, covering compilation, program structure, standard output functions, data types, coding standards, and the execution process of C programs.

## Learning Objectives

By completing this project, the following concepts are explored and practiced:

- Introduction to C programming
- History of C and its creators
- The compilation process using GCC
- Program entry points and the `main` function
- Output functions: `printf`, `puts`, and `putchar`
- Using the `sizeof` operator
- Understanding data types and memory sizes
- GCC compilation basics
- Default executable naming conventions
- C coding standards and style compliance
- Working with header files and standard libraries
- Program exit status and return values

## Technologies Used

- C Programming Language
- GCC Compiler
- Linux/Ubuntu
- Betty Coding Style
